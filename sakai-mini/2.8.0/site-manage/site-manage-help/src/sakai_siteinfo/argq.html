Site Info

Overview

The Site Info tool provides information about the site that you are currently in. If you have a role that allows it, you can use this tool to make changes to information about the site, tools available in the site, and access to the site. You can also publish the site using the Site Info tool. More...

How do I?

  • Adding, editing, removing, hiding, or reordering tools
  • Adding, editing, or removing participants
  • Viewing participants in your site
  • Changing participant roles
  • Managing groups
  • Controlling access to your site
  • Editing information about your site
  • Publishing or unpublishing a site
  • Changing your site's appearance
  • Importing material or participants from another site you own
  • Viewing site information
  • Duplicating a site
  • Adding or removing a class or section
  • Editing your site information if you've removed the Site Info tool
